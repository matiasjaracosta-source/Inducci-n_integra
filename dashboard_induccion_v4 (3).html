⬆ ¿Cómo actualizar los datos del dashboard?

1Actualiza el registro en el Excel (Piloto_Induccion_v5) con los nuevos datos o cambios de estado.
2Guarda el Excel y súbelo en el chat con el mensaje: "Necesito actualizar el dashboard HTML".
3Recibirás un nuevo archivo HTML con los datos actualizados, los segmentadores ajustados y todo listo para usar.
4Descarga el nuevo HTML y reemplaza el archivo anterior.
💡 El dashboard es estático — los datos viven dentro del archivo HTML. Cada vez que subas el Excel actualizado, se genera una nueva versión del dashboard en segundos.
